# stacked_disks — SasView 6.1.0 documentation

Model: stacked_disks
URL: https://www.sasview.org/docs/user/models/stacked_disks.html

## Description
Form factor for a stacked set of non exfoliated core/shell disks

## Parameters
- **scale**: Scale factor or Volume fraction
  Units: None
- **background**: Source background
  Units: cm-1
- **thick_core**: Thickness of the core disk
  Units: Å
- **thick_layer**: Thickness of layer each side of core
  Units: Å
- **radius**: Radius of the stacked disk
  Units: Å
- **n_stacking**: Number of stacked layer/core/layer disks
  Units: None
- **sigma_d**: Sigma of nearest neighbor spacing
  Units: Å
- **sld_core**: Core scattering length density
  Units: 10-6Å-2
- **sld_layer**: Layer scattering length density
  Units: 10-6Å-2
- **sld_solvent**: Solvent scattering length density
  Units: 10-6Å-2
- **theta**: Orientation of the stacked disk axis w/respect incoming beam
  Units: degree
- **phi**: Rotation about beam
  Units: degree

## Equations
\[I(q) = N\int_{0}^{\pi /2}\left[ \Delta \rho_t
\left( V_t f_t(q,\alpha) - V_c f_c(q,\alpha)\right) + \Delta
\rho_c V_c f_c(q,\alpha)\right]^2 S(q,\alpha)\sin{\alpha}\ d\alpha
+ \text{background}\]
\[\Delta \rho_i = \rho_i - \rho_\text{solvent}\]
\[ \begin{align}\begin{aligned}f_t(q,\alpha) =
\left(\frac{\sin(q(d+h)\cos{\alpha})}{q(d+h)\cos{\alpha}}\right)
\left(\frac{2J_1(qR\sin{\alpha})}{qR\sin{\alpha}} \right)\\f_c(q,\alpha) =
\left(\frac{\sin(qh)\cos{\alpha})}{qh\cos{\alpha}}\right)
\left(\frac{2J_1(qR\sin{\alpha})}{qR\sin{\alpha}}\right)\end{aligned}\end{align} \]
\[S(q,\alpha) = 1 + \frac{1}{2}\sum_{k=1}^n(n-k)\cos{(kDq\cos{\alpha})}
\exp\left[ -k(q)^2(D\cos{\alpha}~\sigma_d)^2/2\right]\]
