# hardsphere — SasView 6.1.0 documentation

Model: hardsphere
URL: https://www.sasview.org/docs/user/models/hardsphere.html

## Description
Hard sphere structure factor, with Percus-Yevick closure

## Parameters
- **radius_effective**: effective radius of hard sphere
  Units: Å
- **volfraction**: volume fraction of hard spheres
  Units: None

## Equations
\[\begin{split}U(r) = \begin{cases}
\infty & r < 2R \\
0 & r \geq 2R
\end{cases}\end{split}\]
\[q = \sqrt{q_x^2 + q_y^2}\]
