# hollow_rectangular_prism — SasView 6.1.0 documentation

Model: hollow_rectangular_prism
URL: https://www.sasview.org/docs/user/models/hollow_rectangular_prism.html

## Description
Hollow rectangular parallelepiped with uniform scattering length density.

## Parameters
- **scale**: Scale factor or Volume fraction
  Units: None
- **background**: Source background
  Units: cm-1
- **sld**: Parallelepiped scattering length density
  Units: 10-6Å-2
- **sld_solvent**: Solvent scattering length density
  Units: 10-6Å-2
- **length_a**: Shortest, external, size of the parallelepiped
  Units: Å
- **b2a_ratio**: Ratio sides b/a
  Units: Å
- **c2a_ratio**: Ratio sides c/a
  Units: Å
- **thickness**: Thickness of parallelepiped
  Units: Å
- **theta**: c axis to beam angle
  Units: degree
- **phi**: rotation about beam
  Units: degree
- **psi**: rotation about c axis
  Units: degree

## Equations
\[P(q) =  \frac{1}{V^2} \frac{2}{\pi} \times \, \int_0^{\frac{\pi}{2}} \,
\int_0^{\frac{\pi}{2}} A_{P\Delta}^2(q) \, \sin\theta \, d\theta \, d\phi\]
\begin{align*}
A_{P\Delta}(q) & =  A B C
  \left[\frac{\sin \bigl( q \frac{C}{2} \cos\theta \bigr)}
  {\left( q \frac{C}{2} \cos\theta \right)} \right]
  \left[\frac{\sin \bigl( q \frac{A}{2} \sin\theta \sin\phi \bigr)}
  {\left( q \frac{A}{2} \sin\theta \sin\phi \right)}\right]
  \left[\frac{\sin \bigl( q \frac{B}{2} \sin\theta \cos\phi \bigr)}
  {\left( q \frac{B}{2} \sin\theta \cos\phi \right)}\right] \\
  & {} - A' B' C'
  \left[ \frac{\sin \bigl[ q \frac{C'}{2} \cos\theta \bigr]}
  {q \frac{C'}{2} \cos\theta} \right]
  \left[ \frac{\sin \bigl[ q \frac{A'}{2} \sin\theta \sin\phi \bigr]}
  {q \frac{A'}{2} \sin\theta \sin\phi} \right]
  \left[ \frac{\sin \bigl[ q \frac{B'}{2} \sin\theta \cos\phi \bigr]}
  {q \frac{B'}{2} \sin\theta \cos\phi} \right]
\end{align*}
\[I(q) = \text{scale} \times V \times (\rho_\text{p} -
\rho_\text{solvent})^2 \times P(q) + \text{background}\]
