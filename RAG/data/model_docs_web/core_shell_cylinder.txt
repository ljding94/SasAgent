# core_shell_cylinder — SasView 6.1.0 documentation

Model: core_shell_cylinder
URL: https://www.sasview.org/docs/user/models/core_shell_cylinder.html

## Description
Right circular cylinder with a core-shell scattering length density profile.

## Parameters
- **scale**: Scale factor or Volume fraction
  Units: None
- **background**: Source background
  Units: cm-1
- **sld_core**: Cylinder core scattering length density
  Units: 10-6Å-2
- **sld_shell**: Cylinder shell scattering length density
  Units: 10-6Å-2
- **sld_solvent**: Solvent scattering length density
  Units: 10-6Å-2
- **radius**: Cylinder core radius
  Units: Å
- **thickness**: Cylinder shell thickness
  Units: Å
- **length**: Cylinder length
  Units: Å
- **theta**: cylinder axis to beam angle
  Units: degree
- **phi**: rotation about beam
  Units: degree

## Equations
\[I(q,\alpha) = \frac{\text{scale}}{V_s} F^2(q,\alpha).sin(\alpha) + \text{background}\]
\[\begin{split}F(q,\alpha) = &\ (\rho_c - \rho_s) V_c
       \frac{\sin \left( q \tfrac12 L\cos\alpha \right)}
            {q \tfrac12 L\cos\alpha}
       \frac{2 J_1 \left( qR\sin\alpha \right)}
            {qR\sin\alpha} \\
     &\ + (\rho_s - \rho_\text{solv}) V_s
       \frac{\sin \left( q \left(\tfrac12 L+T\right) \cos\alpha \right)}
            {q \left(\tfrac12 L +T \right) \cos\alpha}
       \frac{ 2 J_1 \left( q(R+T)\sin\alpha \right)}
            {q(R+T)\sin\alpha}\end{split}\]
\[V_s = \pi (R + T)^2 (L + 2T)\]
