# peak_lorentz — SasView 6.1.0 documentation

Model: peak_lorentz
URL: https://www.sasview.org/docs/user/models/peak_lorentz.html

## Parameters
- **scale**: Scale factor or Volume fraction
  Units: None
- **background**: Source background
  Units: cm-1
- **peak_pos**: Peak postion in q
  Units: Å-1
- **peak_hwhm**: HWHM of peak
  Units: Å-1

## Equations
\[I(q) = \frac{scale}{\bigl(1+\bigl(\frac{q-q_0}{B}\bigr)^2\bigr)} + background\]
\[q = \sqrt{q_x^2 + q_y^2}\]
