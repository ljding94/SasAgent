# core_shell_bicelle — SasView 6.1.0 documentation

Model: core_shell_bicelle
URL: https://www.sasview.org/docs/user/models/core_shell_bicelle.html

## Description
Circular cylinder with a core-shell scattering length density profile..

## Parameters
- **scale**: Scale factor or Volume fraction
  Units: None
- **background**: Source background
  Units: cm-1
- **radius**: Cylinder core radius
  Units: Å
- **thick_rim**: Rim shell thickness
  Units: Å
- **thick_face**: Cylinder face thickness
  Units: Å
- **length**: Cylinder length
  Units: Å
- **sld_core**: Cylinder core scattering length density
  Units: 10-6Å-2
- **sld_face**: Cylinder face scattering length density
  Units: 10-6Å-2
- **sld_rim**: Cylinder rim scattering length density
  Units: 10-6Å-2
- **sld_solvent**: Solvent scattering length density
  Units: 10-6Å-2
- **theta**: cylinder axis to beam angle
  Units: degree
- **phi**: rotation about beam
  Units: degree

## Equations
\[\begin{split}\rho(r) =
  \begin{cases}
  &\rho_c \text{ for } 0 \lt r \lt R; -L \lt z\lt L \\[1.5ex]
  &\rho_f \text{ for } 0 \lt r \lt R; -(L+2t) \lt z\lt -L;
  L \lt z\lt (L+2t) \\[1.5ex]
  &\rho_r\text{ for } 0 \lt r \lt R; -(L+2t) \lt z\lt -L; L \lt z\lt (L+2t)
  \end{cases}\end{split}\]
\[I(Q,\alpha) = \frac{\text{scale}}{V_t} \cdot
    F(Q,\alpha)^2 \cdot sin(\alpha) + \text{background}\]
\begin{align*}
F(Q,\alpha) = &\bigg[
(\rho_c - \rho_f) V_c
 \frac{2J_1(QRsin \alpha)}{QRsin\alpha}
 \frac{sin(QLcos\alpha/2)}{Q(L/2)cos\alpha} \\
&+(\rho_f - \rho_r) V_{c+f}
 \frac{2J_1(QRsin\alpha)}{QRsin\alpha}
 \frac{sin(Q(L/2+t_f)cos\alpha)}{Q(L/2+t_f)cos\alpha} \\
&+(\rho_r - \rho_s) V_t
 \frac{2J_1(Q(R+t_r)sin\alpha)}{Q(R+t_r)sin\alpha}
 \frac{sin(Q(L/2+t_f)cos\alpha)}{Q(L/2+t_f)cos\alpha}
\bigg]
\end{align*}
