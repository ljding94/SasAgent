# stickyhardsphere — SasView 6.1.0 documentation

Model: stickyhardsphere
URL: https://www.sasview.org/docs/user/models/stickyhardsphere.html

## Description
‘Sticky’ hard sphere structure factor with Percus-Yevick closure

## Parameters
- **radius_effective**: effective radius of hard sphere
  Units: Å
- **volfraction**: volume fraction of hard spheres
  Units: None
- **perturb**: perturbation parameter, tau
  Units: None
- **stickiness**: stickiness, epsilon
  Units: None

## Equations
\[\begin{split}\epsilon     &= \frac{1}{12\tau} \exp(u_o / kT) \\
\tau &= \Delta / (\sigma + \Delta)\end{split}\]
\[\begin{split}U(r) = \begin{cases}
    \infty & r < \sigma \\
    -U_o   & \sigma \leq r \leq \sigma + \Delta \\
    0      & r > \sigma + \Delta
    \end{cases}\end{split}\]
\[q = \sqrt{q_x^2 + q_y^2}\]
