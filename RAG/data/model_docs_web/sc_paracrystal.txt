# sc_paracrystal — SasView 6.1.0 documentation

Model: sc_paracrystal
URL: https://www.sasview.org/docs/user/models/sc_paracrystal.html

## Description
Simple cubic lattice with paracrystalline distortion

## Parameters
- **scale**: Scale factor or Volume fraction
  Units: None
- **background**: Source background
  Units: cm-1
- **dnn**: Nearest neighbor distance
  Units: Å
- **d_factor**: Paracrystal distortion factor
  Units: None
- **radius**: Radius of sphere
  Units: Å
- **sld**: Sphere scattering length density
  Units: 10-6Å-2
- **sld_solvent**: Solvent scattering length density
  Units: 10-6Å-2
- **theta**: c axis to beam angle
  Units: degree
- **phi**: rotation about beam
  Units: degree
- **psi**: rotation about c axis
  Units: degree

## Equations
\[I(q) = \frac{\text{scale}}{V_p} V_\text{lattice} P(q) Z(q) + \text{background}\]
\[\begin{split}\vec{a_1} &= \vec{b_1} = a \hat{\textbf{x}} \\
\vec{a_2} &= \vec{b_2} = a \hat{\textbf{y}} \\
\vec{a_3} &= \vec{b_3} = a \hat{\textbf{z}}.\end{split}\]
\[V_{lattice} = \frac{4/3 \pi R^3}{D^3}\]
\[Z(\vec{q}) = \prod_{k=1}^{3}Z_k(\vec{q})\]
\[Z_k(\vec{q}) = \frac{1-|F_k|^2}{1-2|F_k|\cos(\vec{a_k}\cdot\vec{q})+|F_k|^2}\]
\[F_k(\vec{q}) = e^{-\frac{1}{2} \Delta a^2_k q^2} \times e^{-i\vec{q}\cdot\vec{a_k}}.\]
\[\begin{split}Z_1(q,\theta,\phi)&=[1-e^{-q^2\Delta a^2}]/\{1-2e^{-\frac{1}{2}q^2\Delta a^2}\cos(-qD \sin\theta \cos\phi ) + e^{-q^2\Delta a^2}\}\\
Z_2(q,\theta,\phi)&=[1-e^{-q^2\Delta a^2}]/\{1-2e^{-\frac{1}{2}q^2\Delta a^2}\cos(qD \sin\theta \sin\phi) + e^{-q^2\Delta a^2}\}\\
Z_3(q,\theta,\phi)&=[1-e^{-q^2\Delta a^2}]/\{1-2e^{-\frac{1}{2}q^2\Delta a^2}\cos(qD \cos\theta) + e^{-q^2\Delta a^2}\}.\\\end{split}\]
\[\frac{qa}{2\pi}=\frac{qD}{2\pi} = \sqrt{h^2+k^2+l^2}\]
