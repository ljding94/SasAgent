# flexible_cylinder_elliptical — SasView 6.1.0 documentation

Model: flexible_cylinder_elliptical
URL: https://www.sasview.org/docs/user/models/flexible_cylinder_elliptical.html

## Description
Flexible cylinder wth an elliptical cross section and a uniform scattering length density.

## Parameters
- **scale**: Scale factor or Volume fraction
  Units: None
- **background**: Source background
  Units: cm-1
- **length**: Length of the flexible cylinder
  Units: Å
- **kuhn_length**: Kuhn length of the flexible cylinder
  Units: Å
- **radius**: Radius of the flexible cylinder
  Units: Å
- **axis_ratio**: Axis_ratio (major_radius/minor_radius
  Units: None
- **sld**: Cylinder scattering length density
  Units: 10-6Å-2
- **sld_solvent**: Solvent scattering length density
  Units: 10-6Å-2

## Equations
\[P(q) = \text{scale} \left<F^2\right>/V + \text{background}\]
\[q = \sqrt{q_x^2 + q_y^2}\]
