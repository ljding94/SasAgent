## Equations
\[I(q) = \text{scale} / q + \text{background}\]
\[I(q) = \text{background}
+ \sum_{i=1}^N \Bigl[
    G_i \exp\Bigl(-\frac{q^2R_{gi}^2}{3}\Bigr)
   + B_i \exp\Bigl(-\frac{q^2R_{g(i+1)}^2}{3}\Bigr)
         \Bigl(\frac{1}{q_i^*}\Bigr)^{P_i} \Bigr]\]
\[q_i^* = q \left[\operatorname{erf}
        \left(\frac{q R_{gi}}{\sqrt{6}}\right)
    \right]^{-3}\]
\[q = \sqrt{q_x^2 + q_y^2}\]
