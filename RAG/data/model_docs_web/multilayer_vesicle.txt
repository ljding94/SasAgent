# multilayer_vesicle — SasView 6.1.0 documentation

Model: multilayer_vesicle
URL: https://www.sasview.org/docs/user/models/multilayer_vesicle.html

## Parameters
- **scale**: Scale factor or Volume fraction
  Units: None
- **background**: Source background
  Units: cm-1
- **volfraction**: volume fraction of vesicles
  Units: None
- **radius**: radius of solvent filled core
  Units: Å
- **thick_shell**: thickness of one shell
  Units: Å
- **thick_solvent**: solvent thickness between shells
  Units: Å
- **sld_solvent**: solvent scattering length density
  Units: 10-6Å-2
- **sld**: Shell scattering length density
  Units: 10-6Å-2
- **n_shells**: Number of shell plus solvent layer pairs (must be integer)
  Units: None

## Equations
\[P(q) = \text{scale} \cdot \frac{\phi}{V(R_N)} F^2(q) + \text{background}\]
\[F(q) = (\rho_\text{shell}-\rho_\text{solv}) \sum_{i=1}^{N} \left[
3V(r_i)\frac{\sin(qr_i) - qr_i\cos(qr_i)}{(qr_i)^3}
- 3V(R_i)\frac{\sin(qR_i) - qR_i\cos(qR_i)}{(qR_i)^3}
\right]\]
\[\begin{split}r_i &= r_c + (i-1)(t_s + t_w) \text{ solvent radius before shell } i \\
R_i &= r_i + t_s \text{ shell radius for shell } i\end{split}\]
\[q = \sqrt{q_x^2 + q_y^2}\]
