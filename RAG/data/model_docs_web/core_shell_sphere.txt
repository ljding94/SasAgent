# core_shell_sphere — SasView 6.1.0 documentation

Model: core_shell_sphere
URL: https://www.sasview.org/docs/user/models/core_shell_sphere.html

## Description
Form factor for a monodisperse spherical particle with particle with a core-shell structure.

## Parameters
- **scale**: Scale factor or Volume fraction
  Units: None
- **background**: Source background
  Units: cm-1
- **radius**: Sphere core radius
  Units: Å
- **thickness**: Sphere shell thickness
  Units: Å
- **sld_core**: core scattering length density
  Units: 10-6Å-2
- **sld_shell**: shell scattering length density
  Units: 10-6Å-2
- **sld_solvent**: Solvent scattering length density
  Units: 10-6Å-2

## Equations
\[P(q) = \frac{\text{scale}}{V} F^2(q) + \text{background}\]
\[F(q) = 3\left[
   V_c(\rho_c-\rho_s)\frac{\sin(qr_c)-qr_c\cos(qr_c)}{(qr_c)^3} +
   V(\rho_s-\rho_\text{solv})\frac{\sin(qr_s)-qr_s\cos(qr_s)}{(qr_s)^3}
   \right]\]
