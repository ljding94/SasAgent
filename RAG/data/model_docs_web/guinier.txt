# guinier — SasView 6.1.0 documentation

Model: guinier
URL: https://www.sasview.org/docs/user/models/guinier.html

## Parameters
- **scale**: Scale factor or Volume fraction
  Units: None
- **background**: Source background
  Units: cm-1
- **rg**: Radius of Gyration
  Units: Å

## Equations
\[I(q) = \text{scale} \cdot \exp{\left[ \frac{-Q^2 R_g^2 }{3} \right]}
        + \text{background}\]
\[q = \sqrt{q_x^2 + q_y^2}\]
\[R_g^2 = \frac{\sum_i\rho_i\left(r_i-r_0\right)^2}{\sum_i\rho_i}\]
